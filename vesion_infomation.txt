TOA 학원 추천 AI: 버전 1.0 문서
문서명: ATOA 학원 추천 AI 프로젝트 버전 1.0 관리 문서
버전: 1.0
작성일: 2025년 8월 14일
개요:
ATOA 학원 추천 AI 프로젝트의 첫 번째 버전인 1.0은 지도 기반의 학원 검색 및 정보 표시 기능을 구현하고, 향후 AI 기능이 통합될 기반을 마련하는 데 중점을 두었습니다. 사용자는 지도를 통해 학원을 탐색하고, 간단한 검색 및 필터링 기능을 활용할 수 있습니다. 또한, AI 기능 도입을 위한 채팅 UI를 완성하여 대화형 서비스의 기초를 구축했습니다.

주요 기능
지도 기반 학원 검색: 사용자가 지도를 이동하거나 확대/축소하면, 현재 지도 화면에 보이는 범위 내의 학원 정보를 자동으로 불러와 마커로 표시합니다.

검색 및 필터링: 학원명 키워드와 교습과정을 선택하여 학원 목록을 필터링할 수 있습니다.

학원 상세 정보: 지도상의 마커를 클릭하면, 해당 학원의 상세 정보(학원명, 주소, 전화번호, 교습과정)를 보여주는 팝업이 나타납니다.

사용자 인터페이스 (UI/UX)
AI 기능 팝업: 'AI 기능' 버튼을 클릭하면 새로운 팝업 창이 나타납니다.

대화형 채팅 UI: 팝업 내부에 AI와 사용자의 대화를 보여주는 채팅창, 질문 입력창, 그리고 키워드 선택 버튼들이 구현되었습니다.

키워드 선택: 사용자는 미리 정의된 키워드 버튼을 클릭하여 질문에 키워드를 추가할 수 있으며, 선택된 키워드는 해시태그 형태로 표시됩니다.

시뮬레이션 응답: 사용자가 질문을 전송하면, 미리 정의된 10개의 답변 중 하나를 무작위로 선택하여 마치 AI가 답변하는 것처럼 시뮬레이션합니다.

기술 스택
프론트엔드: HTML, CSS, JavaScript

지도 서비스: 카카오맵(Kakao Maps) API

백엔드: Firebase Functions (노드.js) 및 Firestore

API: Firebase Functions를 통해 백엔드 API 엔드포인트(예: /api/academies, /api/ai-chat)를 구축했습니다.
